# Student Record Management System

## Overview
This project was created as my final assignment for Computer Programming II. It is a menu-driven C++ program that manages basic student records.

## Features
The program allows the user to add new student records and display all stored student information. Each record includes a student ID, name, age, and marks.

## Technologies Used
It was built using fundamental C++ concepts such as classes, functions, loops, and user input handling. The goal of this project is to demonstrate understanding of basic object-oriented programming and program structure.

## How to Run
Compile and run the program using a C++ compiler.
